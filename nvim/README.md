# Neovim Config

### Install Neovim

```sh
brew install nvim
```

### Clone this config

```sh
git clone https://github.com/rush1016/nvim-config.git ~/.config/nvim
```

### Start nvim to install the plugins

```sh
nvim
```
### (Optional) Install LazyGit

```sh
brew install lazygit
```
